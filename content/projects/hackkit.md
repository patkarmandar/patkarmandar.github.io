HackKit (The Hacker Kit) is a web-based, curated directory of cybersecurity tools designed for security researchers, penetration testers, and learners.

It provides a clean interface to discover, search, and filter security tools across multiple categories, making it easier to quickly find the right tool for a specific security task.

### Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- UI Framework: Bootstrap 5
- Icons: Bootstrap Icons
- Hosting: Static (GitHub Pages compatible)

### Highlights
- Focus on usability and discoverability
- No backend or database required
- Easily extensible by adding new tools or tags
